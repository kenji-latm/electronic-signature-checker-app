# トウシテナ（登記用電子署名チェッカー）

商業・法人登記で使う電子署名・電子証明書について、法務省掲載一覧との照合を行う静的Webアプリです。

## 機能

- 電子署名サービス名、証明書名、発行者名で検索
- 申請書情報、委任状情報、添付書面情報などの利用場面を表示
- 署名付きPDFから署名者候補、証明書発行者、有効期間を読み取り
- PDF内の証明書情報を法務省掲載一覧と照合
- 法務省ページの掲載一覧を更新スクリプトで取得

## 注意

このアプリのPDFチェックは事前確認です。失効確認、認証局チェーンの完全検証、登記・供託オンライン申請システムと同等の最終判定を保証するものではありません。

## データ更新

```powershell
npm run scrape
```

または Windows では `更新.bat` を実行します。
